--Installation--

Just download the latest release and extecute it.

--Controls--

Movement

A - Left
D - Right
Spacebar - jump
Left shift - dash
 

Debbuging tools

F1 - Restart Game
F2 - Restart level
F3 - Change level
F5 - Quicksave
F6 - Load
F9 - Show Colliders
F10 - Godmode
